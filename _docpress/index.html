Bienvenido a mi sitio de documentación

Este es un proyecto hecho con DocPress.

DocPress es una herramienta que sirve para generar micrositios de documentación.

Nosotros escribimos la documentación en formato markdown y DocPress la convierte a código HTML.

Para instalar DocPress necesitamos tener instalado Node.js.
